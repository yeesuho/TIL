### 2. 캡슐화(접근지정자 private)

```java
class Parent {

private int num = 10;


public set_num(int m) {
    this.num+=m
}
public get_num() {
    return this.num
}
}

class child extends parent {

    int num++;
}

// 메인함수에 객체 만들고 child.num++
```
