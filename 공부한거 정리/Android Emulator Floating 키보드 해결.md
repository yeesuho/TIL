아니 제가 기능대회 문제를 풀면서 안드로이드 모바일 에뮬레이터 키보드가 자꾸 킹받게 Floating으로 뜨는데 해제가 안되는겁니다... 이게 더 킹받는건 비밀번호 TextField는 일반 키보드로 잘 뜨는데 이메일 입력 필드는 Floating 해제하는것도 없고 그냥 못생긴(주관적인 생각) Floating 키보드를 보여줘서 열심히 구글링을 해본결과 밑에 해결법을 따라했더니 됐습니다 근데 저게 정답인지는 저도 모름 저는 일단 해결됐어요

### 1단계
Settings -> System -> Keyboard -> On-screen keyboard -> Gboard

### 2단계
**Physical keyboard -> turn on "Show on-screen keyboard"**

### 3단계
**Go back one level (to Gboard's setting) -> Write in text fields -> turn off "Use stylus to write in text fields"**

네 저 같은 상황이라면 알아서 잘 따라하시면 됩니다 <br>
